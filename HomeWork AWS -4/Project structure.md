# AWS VPC + EC2 Terraform Setup

## Описание

Этот проект разворачивает следующую инфраструктуру в AWS с помощью Terraform:

- VPC с двумя подсетями: публичной и приватной
- Internet Gateway и NAT Gateway
- EC2-инстанс в публичной подсети с доступом из интернета
- EC2-инстанс в приватной подсети, доступный только из публичного инстанса
- SSH-ключ генерируется автоматически
- Все ресурсы удаляются через `terraform destroy`
